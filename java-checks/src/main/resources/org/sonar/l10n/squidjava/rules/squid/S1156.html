Concatenation between primitives or Objects and empty strings are done to benefit from the implicit typecast to String. While this works, it is not as efficient as calling String.valueOf(primitive) or Object.toString() directly.

The following code:

private static final int DEFAULT_VALUE = 3;
private int String value = "" + DEFAULT_VALUE;              // Non-Compliant

should be refactored into:

private static final int DEFAULT_VALUE = 3;
private int String value = String.valueOf(DEFAULT_VALUE);   // Compliant

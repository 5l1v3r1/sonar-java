Each source file should always start with a header stating who owns the file and the license which must be used to distribute the application. This rule must be fed with a regular expression which is used to be sure that the beginning of the file start with the expected header.

For example, the header format regular expression can be set to:

/\*
 \* (C) Copyright 20\d\d Nuxeo SA (http://nuxeo\.com/) and others\.
 \*
 \* All rights reserved\. This program and the accompanying materials
 \* are made available under the terms of the GNU Lesser General Public License
 \* (LGPL) version 2\.1 which accompanies this distribution, and is available at
 \* http://www\.gnu\.org/licenses/lgpl-2\.1\.html
 \*
 \* This library is distributed in the hope that it will be useful,
 \* but WITHOUT ANY WARRANTY; without even the implied warranty of
 \* MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE\. See the GNU
 \* Lesser General Public License for more details\.
 \*/

Sharing some coding conventions is a key point to make it possible for a team to efficiently collaborate. This rule make it mandatory to place any curly braces closing a block of code on a dedicated line of code.

The following code snippet illustrates this rule: 

public void myMethod() {
  if(something) {
    executeTask();}                   // Non-Compliant
  else if (somethingElse) {
    doSomethingElse();
  }                                   // Compliant

  try {
    generateOrder();
  } finally {                         // Compliant
    closeConnection();
  }                                   // Compliant

  try {
    generateOrder();
  } finally {                         // Compliant
    closeConnection(); }              // Non-Compliant
}

Early classes of the Java Collections Framework, such as Vector and Hashtable, were synchronized to make them thread-safe. Unfortunately, synchronization has a big negative impact on performance, even when using these collections from a single thread.

It is better to use their new unsynchronized replacements:

  • ArrayList or LinkedList instead of Vector
  • HashMap instead of Hashtable

The following code:

Vector cats = new Vector();       // Non-Compliant

should be refactored into:

ArrayList cats = new ArrayList(); // Compliant

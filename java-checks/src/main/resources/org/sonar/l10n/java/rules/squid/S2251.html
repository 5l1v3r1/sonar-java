A for loop with a stop condition that can never be reached will run infinitely. While there are occassions when an infinite loop is intended, the convention is to construct such loops as while loops. More typically, an infinite for loop is a bug.

Noncompliant Code Example

for (int i = 0; i < 10; i--) { // Noncompliant; i will always be <10
  //...
}

Compliant Solution

for (int i = 0; i < 10; i++) {
  //...
}

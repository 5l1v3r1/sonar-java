A String.startsWith test for a single letter can be simplified and made more performant by switching to a comparison of String.charAt(0).

Similarly, an indexOf or lastIndexOf call with a char argument will run faster than one with a String argument.

Noncompliant Code Example

String myStr = "Hello World";
// ...

int pos = myStr.indexOf("W");  // Noncompliant

if (myStr.startsWith("A")) {  // Noncompliant
  // ...
}

Compliant Solution

String myStr = "Hello World";
// ...

int pos = myStr.indexOf('W'); 

if (myStr.length() > 0 && myStr.charAt(0) == 'A') {
  // ...
}

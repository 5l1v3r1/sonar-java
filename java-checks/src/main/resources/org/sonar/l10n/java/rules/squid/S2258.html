By contract, The NullCipher class is a class that provides an "identity cipher" -- one that does not tranform the plaintext. As a consequence, the ciphertext is identical to the plaintext. So this class should mainly be used for testing purpose.

Noncompliant Code Example

NullCipher nc = new NullCipher();

See

  • CWE-327: Use of a Broken or Risky Cryptographic Algorithm
  • OWASP Top Ten 2013 Category A6 - Sensitive Data Exposure

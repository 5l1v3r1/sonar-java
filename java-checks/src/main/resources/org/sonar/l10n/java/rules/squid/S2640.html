When a class implements an Interface, all of its subclasses automatically inherit that implementation, so there's no need to explicitly re-implement a superclass' Interfaces in a child class. Simply @Override the parent class' Interface methods.

Noncompliant Code Example

public class Fruit implements Fiberous {
  @Override
  public int getFiberCount() { ... }
}

public class Raspberry extends Fruit implements Fiberous { // Noncompliant
  @Override
  public int getFiberCount() { ... }
}

Compliant Solution

public class Fruit implements Fiberous {
  @Override
  public int getFiberCount() { ... }
}

public class Raspberry extends Fruit {
  @Override
  public int getFiberCount() { ... }
}
